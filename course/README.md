# Skill Authoring: 0 to Hero

A structured course that takes you from knowing nothing about AI agent skills to authoring production-quality SKILL.md files. Each module builds on the last. Estimated time: 3-4 hours total.

## Prerequisites

- Basic familiarity with Markdown
- Access to an AI coding agent (Claude Code, Cursor, Codex CLI, or similar)
- This repository cloned locally

## Course Map

```
Module 1: What Are Skills?           (30 min)  -- Foundation
Module 2: The SKILL.md Format        (30 min)  -- Spec mastery
Module 3: Writing Descriptions       (30 min)  -- The #1 failure point
Module 4: Progressive Disclosure     (20 min)  -- Token economics
Module 5: Patterns That Work         (30 min)  -- Battle-tested patterns
Module 6: Anti-Patterns              (20 min)  -- What NOT to do
Module 7: Your First Skill           (40 min)  -- Hands-on lab
Module 8: Advanced Techniques        (30 min)  -- Meta-skills, composition, instincts
Module 9: Multi-Host Compatibility   (20 min)  -- Ship everywhere
Module 10: Maintaining a Skill Library (20 min) -- Living systems
```

## How to take this course

Each module is a standalone markdown file. Read them in order. Modules include:
- Concepts with real examples from top repos
- Exercises to practice
- Links to wiki articles for deeper reading

Start with [Module 1: What Are Skills?](01-what-are-skills.md)
